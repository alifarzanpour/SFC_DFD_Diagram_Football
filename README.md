# SFC_DFD_Diagram_Football
نمودارهای مدلسازی وظیفه‌ای و پردازشی روشهای رایج در تحلیل و طراحی سیستمهاست که به منظور  نمایش و درک بهتر جریان اطلاعات، عملیات پردازشی و تعامل میان اجزای مختلف یک سیستم به کار میرود. هدف اصلی این مدلسازی، تجزیه و تحلیل فرآیندهای موجود یا طراحی فرآیندهای جدید به صورت گرافیکی و قابل فهم برای تمامی ذینفعان سیستم است.
